<h2 id="introducao-ao-nestjs"> 💻 Introdução ao NestJS</h2> 

NestJS é um framework para a construção de aplicações Node.js eficientes, confiáveis e escaláveis no lado do servidor. Criado por Kamil Myśliwiec, ele utiliza TypeScript por padrão (embora permita JavaScript puro) e combina elementos de Programação Orientada a Objetos (OOP), Programação Funcional (FP) e Programação Reativa Funcional (FRP). Sob o capô, o Nest usa frameworks HTTP robustos como Express (padrão) ou Fastify, abstraindo sua complexidade e oferecendo um nível muito mais alto de organização e modularidade. 

<h2 id="porque-usar-nestjs"> 📌 Por que usar NestJS?</h2> 

- **Arquitetura Pronta e Padronizada**: Fornece uma arquitetura de software sólida (fortemente inspirada no Angular) que ajuda as equipes a criar código altamente testável, escalável e de fácil manutenção. 
- **Suporte Nativo ao TypeScript**: Garante tipagem forte, auto-completar aprimorado e uma experiência de desenvolvimento superior. 
- **Injeção de Dependência**: Possui um poderoso contêiner de Injeção de Dependência (DI) embutido, facilitando o desacoplamento de componentes e a escrita de testes. 
- **Ecossistema Rico e Versátil**: Integração fácil e "out-of-the-box" com diversas ferramentas e bibliotecas, como TypeORM, Mongoose, GraphQL, WebSockets, Redis, filas de mensagens, entre outros.
